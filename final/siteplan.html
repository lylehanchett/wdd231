DLB Integrators – Website Plan

Site Name

DLB Integrators

This name represents the company and the work performed. DLB Integrators specializes in integrating new automated equipment into existing production lines, building custom equipment to support automation, and providing installation and service for all automated machinery. The name is accurate, professional, and directly tied to the services offered.

Site Purpose

The purpose of this website is to provide an overview of automation integration services offered by DLB Integrators. The site will highlight expertise in integrating new automated machines, designing and fabricating custom stainless-steel equipment, and installing and servicing automation systems. Visitors will be able to explore custom equipment options, learn about available installation services, and view dynamically loaded automation components using modern web technologies.

Scenarios

  • “What types of custom stainless-steel equipment can DLB Integrators build for my automation project?”
  • “How can DLB Integrators help automate my current production line?”
  • “What installation services are available for integrating new machines into my facility?”

Color Scheme

The following colors will be used throughout the website:

#123a63 – Primary Color

Used for the header, footer, headings, and key UI elements.

#f5b942 – Accent Color

Used for highlights, borders, buttons, and emphasis.

Typography

Font Family: Segoe UI, sans-serif

  • Headings (h1–h4): Segoe UI – bold
  • Body Text: Segoe UI – regular
  • Navigation: Segoe UI – bold

The font was selected for its clarity, readability, and modern professional appearance.
